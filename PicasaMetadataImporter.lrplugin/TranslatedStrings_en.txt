Instead this plug-in extracts Picasa metadata from one or more picasa.ini files created by Picasa and writes the metadata to the Lightroom catalogue using the Lightroom API.^r^rThe import process is a one way import from Picasa to Lightroom.  There is no undo.  Backup your catalogue.^r^rThe plug-in uses filenames to match a file in Picasa to a file in Lightroom.  If different files have the same name, metadata will be written to the wrong file.  If the same file has different filename the plug-in will not see the file and report it unavailable."
